Tumblr photo feed viewer built with Swift and AFNetworking, with progressive image loading.

## Tech

| Layer | Tech |
|---|---|
| Language | Swift |
| UI | UIKit, UITableView |
| Networking | AFNetworking (CocoaPods) |

## Setup

```bash
git clone https://github.com/gerardrecinto/tumblr-feed-ios.git
cd tumblr-feed-ios
pod install
open tumblr.xcworkspace
```

Build and run on the iOS Simulator (Xcode 8+).
