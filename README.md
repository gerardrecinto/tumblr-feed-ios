# Tumblr Feed Lab

![Swift](https://img.shields.io/badge/Swift-3%2B-F05138?logo=swift&logoColor=white)
![iOS 9+](https://img.shields.io/badge/iOS-9%2B-000000?logo=apple&logoColor=white)
![UIKit](https://img.shields.io/badge/UIKit-UITableView-blue)

iOS Tumblr feed viewer built with Swift and UIKit. Fetches posts from the Tumblr API and displays them in a table view with progressive image loading.

## Tech

| Layer | Tech |
|---|---|
| Language | Swift |
| UI | UIKit, UITableView |
| Networking | AFNetworking (CocoaPods) |

## Setup

```bash
cd lab1
pod install
open tumblr.xcworkspace
```

Build and run on the iOS Simulator (Xcode 8+).
